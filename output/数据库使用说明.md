### 3. 历年录取数据（两表通用，下标 1/2/3 = 2025/2024/2023）
| 字段 | 说明 |
|---|---|
| `最低分1` `录取人数1`（普通类另有 `最低位次1`） | 2025 年 |
| `最低分2` `录取人数2`（普通类另有 `最低位次2`） | 2024 年 |
| `最低分3` `录取人数3`（普通类另有 `最低位次3`） | 2023 年 |

> 注意：
> - 艺术类历年分数来自"2025 艺体专家版"文件，按 `院校代码+专业名称` 匹配，覆盖约 89%（校考类、新增专业、停招专业可能为空）。取的是物理科类投档线（艺术类历史/物理绝大多数相同）。
> - **位次口径**：`普通类(招生计划2026)` 的 `最低位次1/2/3` 是**全省首选科目（物理类/历史类）一分段累计位次**（已用官方一分段表校验）。艺术类原源文件的"最低位"是**按艺术类别分别统计的类别内位次**，且多为类别兜底/前缀匹配的近似值，与普通类口径不一致、不可直接比较，故**艺术类表已不保留 `最低位次1/2/3` 列**。艺术类要换算位次，请用 `一分段_美术与设计类综合成绩` 等类别一分段表，对照 `艺术类平行组投档线`。
> - 非艺术类另有 `平均分/平均位次/最高分/最高位次/老批次/计划人数结果` 等更细的历年字段，艺术类无对应数据源，故没有这些列。

### 4b. `一分段_美术与设计类综合成绩`

2025–2026 两年美术与设计类**本科线上综合成绩**一分段合并为一张表，按年份顺序排列（先 2025、后 2026）。

| 字段 | 说明 |
|---|---|
| `年份` | 2025 / 2026 |
| `综合成绩` | 综合成绩（含优惠加分，**小数**，REAL 类型，可直接数值比较） |
| `本段人数` | 该综合成绩档人数 |
| `累计人数` | 该综合成绩及以上累计人数 |

年份分布：2025 共 994 行（末档 250.1，累计 12669 人），2026 共 1008 行（末档 236.9，累计 11228 人）。

> **综合分公式**：`综合成绩 = 文化成绩(含政策性加分)×30% + 专业统考成绩×70%`，与 `艺术类平行组投档线.投档线` 同口径，可用本表把考生综合分换算成**全省综合分位次**，再对照投档线做"冲稳保"。
>
> 数据来源：2025 来自湖南考试院综合成绩分段统计表（图片转录，逐行经累计人数交叉校验）；2026 来自官方 PDF《湖南省2026年普通高考美术与设计类综合成绩分段统计表》（程序解析，累计人数 0 误差）。
>
> **口径提示**：2025 官方表在 298.8→298.5 处直接跳档（无 298.7、298.6 两行，系官方表原样），其余各行累计连续。

### 5. `艺术类平行组投档线`

2024–2025 两年湖南考试院**官方**本科提前批艺术类平行组第一次投档线，合并为一张表。粒度为**院校专业组**（平行志愿按院校专业组投档），覆盖全部艺术类别（音乐/舞蹈/美术与设计类/书法/播音/表导演等）。

| 字段 | 说明 |
|---|---|
| `年份` | 2024 / 2025 |
| `科类` | 如 `艺术类(不分科类)` |
| `计划类别` | `艺术类(平行组)` |
| `院校代号` | 4 位院校代码 |
| `院校名称` | |
| `专业组编号` | 如 `103` |
| `专业组名称` | 如 `第103组(美术与设计类)` |
| `投档线` | 该专业组投档**最低综合分**（为空表示线上生源不足） |
| `文化成绩` `语数之和` `语数最高` `外语` `首选科目` `再选最高` `再选次高` `志愿序号` | 投档最低分**同分排序项**（`再选次高` 仅 2024 为空） |
| `备注` | 院校提出的外语语种、单科成绩等投档要求 |

年份分布：2024 共 1873 行（美术与设计类 558），2025 共 1818 行（美术与设计类 555）。

> **综合分公式**（含美术与设计类）：`综合成绩 = 文化成绩(含政策性加分)×30% + 专业统考成绩×70%`；播音与主持类为 `文化 + 统考`。
>
> 用途：用考生综合分直接对照该表历年 `投档线` 做艺术类平行志愿"冲稳保"分档。**注意**：表中无全省"综合分一分段"，无法将综合分换算成全省综合分位次（需另补一分段数据）。

## 二点五、参考资料表字段说明

### `全国一本二本院校清单`

来源：`raw_data/闲鱼数据/全国一本二本大学清单.md`，由 `scripts/load_university_list.py` 解析导入。

```bash
python3 scripts/load_university_list.py
```

> **口径警告**：本表是**历史/民间口径**的汇总（"一本/二本"指各省把高校放在本科第一/第二批录取的结果，非教育部官方定级；全国多数省份已合并本科批次）。**仅供湖南考生宏观参考，不能作为志愿填报依据**，填报请以湖南省教育考试院当年招生计划与投档线（即招生计划/事实数据各表）为准。

| 字段 | 说明 |
|---|---|
| `ID` | 行序号 |
| `院校名称` | 院校名称（已去除 markdown 标记） |
| `分类` | `一本` / `二本` |
| `子类` | 一本：`双一流` / `普通一本`；二本：`公办二本` / `民办二本` / `合作办学二本` |
| `省份` | 仅一本部分按省份归类（31 省市）；二本部分为空 |
| `排名` | 公办二本=软科排名，民办二本=全国排名（文本，可能为空）；其余为空 |
| `排名口径` | `软科排名` / `民办全国排名` / 空 |
| `备注` | 如 `职业本科/艺术类(无软科综合排名)`、`全国排名100+` 等 |

分类分布：一本 579（双一流 147 / 普通一本 432）；二本 730（公办 569 / 民办 159 / 合作办学 2），共 1309 行。

> **注意**：原始 markdown 二本部分宣称民办二本共 275 所，但正文仅完整列出软科/全国排名靠前者及一份「全国排名100+」名单，故本表民办二本实际收录 159 所（未逐一枚举全部）。同一院校在不同省份可能同时属于一本与二本，本表按原文分别收录，不做去重。
>
> **公办二本 569 所**：原始数据源为 596 所，其中混入了一批国家双一流（含 211）高校（系个别省份/中外合作/民族班有二批招生所致）。为避免误导湖南考生，已剔除 27 所双一流（华中农业、云南大学、广西大学、贵州大学、海南大学、内蒙古大学、新疆大学、宁夏大学、青海大学、西藏大学、石河子大学、延边大学、东北农业、四川农业、北京体育、大连海事、山西大学、河南大学、上海海洋、天津工业、天津中医药、成都中医药、成都理工、南京中医药、广州医科、广州中医药、华南农业），它们均已在一本名单中收录；中国石油大学(北京)克拉玛依校区为双一流校区但部分省份确有低批次招生，予以保留。同步已更新校名：浙江科技学院→浙江科技大学、福建工程学院→福建理工大学、新乡医学院→河南医药大学、上海体育学院→上海体育大学、河北中医学院→河北中医药大学。

常用查询：

查某省一本院校（双一流优先）：
```sql
SELECT 院校名称, 子类 FROM 全国一本二本院校清单
WHERE 分类='一本' AND 省份='湖南'
ORDER BY CASE 子类 WHEN '双一流' THEN 0 ELSE 1 END, ID;
```

查公办二本软科排名前 50：
```sql
SELECT 院校名称, CAST(排名 AS INT) AS 软科排名 FROM 全国一本二本院校清单
WHERE 子类='公办二本' AND 排名!=''
ORDER BY CAST(排名 AS INT) LIMIT 50;
```

判断某院校的历史一本/二本身份：
```sql
SELECT 分类, 子类, 省份 FROM 全国一本二本院校清单 WHERE 院校名称='湖南农业大学';
```

## 四、常用查询示例

### 招生计划

查某院校艺术类专业及历年分：
```sql
SELECT 专业名称, 专业组名称, 计划人数, 学费, 最低分1
FROM 招生计划2026_艺术类
WHERE 院校名称 = '湖南师范大学'
ORDER BY 专业组代码;
```

按 2025 投档线（综合分）筛选可报艺术类专业：
```sql
SELECT 院校名称, 专业名称, 最低分1
FROM 招生计划2026_艺术类
WHERE 最低分1 != '' AND CAST(最低分1 AS REAL) BETWEEN 300 AND 310
ORDER BY CAST(最低分1 AS REAL) DESC;
```

> 艺术类无全省可比"位次"列（见前文位次口径说明）。若要按位次做"冲稳保"，用 `一分段_美术与设计类综合成绩` 把综合分换算成全省综合分位次，再对照 `艺术类平行组投档线`。

同一院校同时查普通类与艺术类（跨表，按院校代码）：
```sql
SELECT '普通' AS 类别, 专业名称, 计划人数 FROM 招生计划2026       WHERE 院校代码='4303'
UNION ALL
SELECT '艺术' AS 类别, 专业名称, 计划人数 FROM 招生计划2026_艺术类 WHERE 院校代码='4303';
```

查某专业组内所有专业（聚合文本）：
```sql
SELECT DISTINCT 院校专业组代码, 专业组名称, 专业组计划人数, 组内专业数, 组内专业
FROM 招生计划2026_艺术类
WHERE 院校名称='湖南师范大学' AND 专业组名称 LIKE '第115组%';
```

### 事实数据

查某年物理类本科控制线：
```sql
SELECT 物理类_普通类_文化, 特殊类型_物理类
FROM 录取控制分数线_普通类体育类
WHERE 年份 = 2026 AND 批次 = '本科';
```

查美术与设计类三年本科控制线变化：
```sql
SELECT 年份, 本科_文化, 本科_专业
FROM 录取控制分数线_艺术类
WHERE 类别 = '美术与设计类'
ORDER BY 年份;
```

根据物理类分数查累计人数（换算位次）：
```sql
SELECT 档分, 累计人数
FROM 一分段_物理类
WHERE 年份 = 2026 AND 档分 = '600';
```

查美术与设计类某统考成绩对应累计人数：
```sql
SELECT 成绩, 累计人数
FROM 一分段_美术与设计类
WHERE 年份 = 2026 AND 成绩 = 250;
```

根据综合成绩查全省综合分位次（取≥该分的最小累计人数，兼容缺档）：
```sql
SELECT 综合成绩, 累计人数
FROM 一分段_美术与设计类综合成绩
WHERE 年份 = 2025 AND 综合成绩 <= 305.4
ORDER BY 综合成绩 DESC
LIMIT 1;
```

综合成绩位次两年对照（如综合分 300 分）：
```sql
SELECT 年份, MIN(累计人数) AS 位次
FROM 一分段_美术与设计类综合成绩
WHERE 综合成绩 >= 300
GROUP BY 年份
ORDER BY 年份;
```

招生计划位次与一分段交叉验证（2025 年物理类）：
```sql
SELECT p.院校名称, p.专业名称, p.最低位次1,
       y.档分, y.累计人数
FROM 招生计划2026 p
JOIN 一分段_物理类 y
  ON y.年份 = 2025 AND y.档分 = p.最低分1
WHERE p.科类 = '物理' AND p.最低分1 != '' AND p.最低位次1 != ''
LIMIT 20;
```

按综合分查美术与设计类可冲院校（2025 投档线在综合分附近）：
```sql
SELECT 院校名称, 专业组名称, 投档线
FROM 艺术类平行组投档线
WHERE 年份 = 2025 AND 专业组名称 LIKE '%美术%'
  AND CAST(投档线 AS REAL) BETWEEN 303 AND 308
ORDER BY CAST(投档线 AS REAL);
```

美术与设计类投档线两年对照（同院校专业组）：
```sql
SELECT a.院校名称, a.专业组名称, a.投档线 AS 投档线2024, b.投档线 AS 投档线2025
FROM 艺术类平行组投档线 a
JOIN 艺术类平行组投档线 b
  ON a.院校代号 = b.院校代号 AND a.专业组名称 = b.专业组名称
WHERE a.年份 = 2024 AND b.年份 = 2025 AND a.专业组名称 LIKE '%美术%'
ORDER BY CAST(b.投档线 AS REAL);
```

## 五、注意事项

1. 招生计划表中分数/位次/计划人数等字段以**文本**存储，空值为 `''`；做数值比较时需 `CAST(... AS INT/REAL)` 并先排除空串。事实数据表数值字段为 INTEGER/REAL 类型，可直接比较。
2. 艺术类 `院校代码` 与非艺术类同体系，但**专业组代码、专业代码逐年重排**，不能跨年用代码匹配专业，只能用专业名称。
3. 数据来源：
   - 招生计划：2026 计划（闲鱼版，已与官方指南抽查核对）；艺术类历年分数（2025 艺体专家版）；院校信息（参考版 + 联网补充）。
   - 事实数据（控制线、一分段）：湖南考试院网站官方数据（`事实数据/` 目录），由 `scripts/load_facts.py` 导入。
   - 艺术类平行组投档线：湖南考试院官方 xlsx（`raw_data/湖南考试院网站官方数据/`），由 `scripts/load_art_toudang.py` 导入。
4. 事实数据表使用 `if_exists="replace"` / `DROP TABLE` 导入，**不影响**已有招生计划表；但重复执行会覆盖事实数据表本身。
5. `艺术类平行组投档线` 字段以**文本**存储，数值比较需 `CAST(... AS REAL)`；`投档线` 为空表示该专业组线上生源不足。
